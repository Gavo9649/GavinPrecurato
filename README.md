# Gavin Precurato Engineering Portfolio

Static portfolio website built for GitHub Pages.

## Files

- `index.html` - page content
- `styles.css` - layout and visual design
- `script.js` - smooth scrolling
- `assets/` - resume PDF and project preview images

## Publish on GitHub Pages

1. Create a GitHub repository for the portfolio.
2. Add these files to the repository root.
3. In GitHub, open `Settings` > `Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose `main` and `/root`, then save.

GitHub will publish the site at the Pages URL shown in that settings screen.
